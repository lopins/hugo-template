Allow none-root users

Solve “no space left on device”

ubuntu

sudo vi /etc/docker/daemon.json

{
        "storage-driver": "devicemapper",
        "storage-opts": [
                "dm.basesize=40G"
        ]
}

Mac

Docker -> settings -> Resources -> Disk Image Size
